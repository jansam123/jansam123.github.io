    Constituent-Based Quark Gluon Tagging using Transformers with the ATLAS detector
    ATLAS Public Note 03 November 2023
    The ATLAS collaboration

    Jet constituents provide a more complete description of the radiation pattern inside a jet than observables describing the global jet properties like the number of tracks or the jet width. An approach to tag quark- and gluon-initiated jets using their constituents (Particle Flow Objects) reconstructed in ATLAS is presented. Monte Carlo simulations of proton–proton collisions at center of mass energy of 13 TeV are used. Transformer architectures are used to learn long-range dependencies between jet constituent kinematic variables. Several variations of transformers are studied, and their performance is compared to taggers based on global jet properties and other jet constituent taggers. A new Deep Learning architecture (DeParT) is presented, slightly outperforming other networks. All models are evaluated on simulated data from multiple Monte Carlo generators to study their model-dependence.

    Python

    Python je môj primárny programovací jazyk. Používam ho na analýzu dát, strojové učenie a základný vývoj webových stránok pre prezentáciu dát. Skúsenosti mám s nasledovnými knižnicami: NumPy, SciPy, Pandas, Matplotlib, Scikit-learn, Dash, Plotly.

    Tensorflow

    Schopný vytvárať a trénovať neurónové siete pomocou Tensorflow. Skúsenosti s Keras, ale aj schopnosť používať predtrénované modely. Použitie od spracovania jazyka po tagovanie častíc.

    C++

    Základné znalosti programovania v C/C++. Použité na spracovanie dát pomocou knižnice ROOT. Predovšetkým používané na vedecké výpočty a analýzu dát.

    ROOT

    ROOT je C++ knižnica pre analýzu dát vyvinutá v CERNe pre vedecké účely. Znalosti základného použitia analýzy dát, ako napríklad histogramy, TTrees a spracovanie dát.

    Docker

    Programy obvykle nasadzujem pomocou dockeru. Skúsenosti s docker-compose.

    Linux

    Linux používam ako operačný systém pre vývoj. Znalosti základných bash skriptov.

    Git

    Všetky moje projekty sú verzované pomocou gitu. Používam hlavne GitHub, občas aj GitLab.

    Slurm

    Skúsenosti s používaním Slurm na využívanie HPC výpočtových zdrojov, ako CPU, tak aj GPU farmy. Skúsenosti so spúšťaním, monitorovaním a managmentom jobov.
